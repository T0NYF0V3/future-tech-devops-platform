# Project Architecture

This document describes the architecture of the Future Tech DevOps Platform.

## Components

- AWS VPC
- Public and private networking
- EC2 instances
- Docker containers
- CI/CD pipelines
- AWS Lambda
- DynamoDB
- CloudWatch monitoring

## Infrastructure Strategy

Infrastructure is provisioned using Infrastructure as Code (IaC) principles with AWS CloudFormation.

## Security

- MFA enabled
- IAM least privilege
- Branch protection rules
- SSM Session Manager access
